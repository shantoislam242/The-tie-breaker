The Tiebreaker is an AI-powered decision-making assistant designed to help you navigate complex choices with clarity and confidence.
When you're stuck between options or facing a difficult dilemma, the app uses Gemini AI to analyze your situation from multiple strategic angles.
Key Features:
Three Analysis Modes:
Pros & Cons: A classic breakdown of the advantages and drawbacks of a single decision.
Comparison Matrix: A side-by-side table comparing different options against specific criteria (e.g., comparing "Sushi" vs. "Chinese Food" based on cost, healthiness, and variety).
SWOT Analysis: A strategic look at the Strengths, Weaknesses, Opportunities, and Threats related to your choice.
AI-Generated Verdict: Beyond just listing facts, the app provides a synthesized "Final Verdict" to give you a clear recommendation based on the analysis.
Clean, Professional Interface: Designed with a focus on readability and "editorial" elegance, using high-quality typography and smooth animations.
Exportable Results: You can easily print or save your analysis for future reference.
How it works:
Input your dilemma: Type in the decision you're trying to make.
Choose your method: Select whether you want a simple list, a comparison table, or a strategic SWOT breakdown.
Get Insights: The AI processes your request and builds a detailed report with a final recommendation to help "break the tie."

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
